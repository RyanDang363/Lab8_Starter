# How are graceful degradation and service workers related? 

Graceful degradation means designing an app to start with its full experience and still work reasonably well when some capabilities are unavailable.

Service workers help support this by intercepting network requests and serving cached resources from the browser when the network fails or is too slow. Instead of the app breaking completely offline, it can fall back to previously saved code and data, so users still get a usable experience even without ideal conditions.
